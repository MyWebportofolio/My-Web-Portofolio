Le français du numérique

  • Bienvenue dans "Le français du numérique"!
  • Leçon 1, les bases.
    • L'objectif de cette leçon est de vous familiariser avec des mots ou des notions clés du numérique.
    • Ces expressions sont souvent présentes sur des sites numériques français.
    • Il est fréquent d'utiliser ces sites pour entreprendre des démarches administratives,
    • Déposer votre CV ou encore acheter des timbres en ligne

Vocabulaire du numérique, actions simples

  • Cliquer, Verbe
    • Cliquer sur le bouton... d'une souris 
ou poser le doigt sur un écran pour 
activer une commande. 
Il suffit de cliquer sur l'onglet pour 
voir ce que ce site propose
  • Mot:Ouvrir, Verbe
    • Afficher le contenu de (un élément), 
lancer l'exécution de (un programme).
(contraire fermer).
  • Mot:Enregistrer, Verbe
    • Stocker (des données) en mémoire. 
Sauvegarder.  Enregistrer un fichier sur 
le disque dur.
  • Mot:Continuer, Verbe
    • Ce mot parfois trouvé sur les sites web  
est souvent "dynamique"  Cliquez-le pour 
être dirigé sur la page suivante ou lire 
une page dans son intégralité.  (l'action contraire 
serait de cliquer sur le mot 
"retour" pour aller sur la page précédente.) 
Quelques variantes: En savoir plus
Appuie sur l'onglet Continuer pour aller sur
la page suivante.
  • Mot:Annuler,Verbe
    • Cette commande vous permet d'annuler la 
dernière action que vous avez effectuée.
J'ai ouvert cette fenêtre de dialogue 
par erreur. Appuie juste sur la commande annuler
  • Mot:Télécharger, Verbe
    • Copier des données (stockées sur un ordinateur 
ou sur un serveur) sur son propre ordinateur par 
l'intermédiaire d'un réseau téléinformatique.
Aujourd'hui je dois télécharger ce document 
sur mon ordinateur

Vocabulaire du numérique, navigation

  • Mot:Page d'accueil, nom féminin
    • Première page d'un site internet
Cette page d'accueil contient beaucoup
d'images
  • Mot:Menu, nom masculin
    • Liste d'options, d'opérations proposées 
à l'utilisateur, sur l'écran d'un ordinateur.
Ouvre le menu déroulant et tu pourras facilement
accéder à une catégorie.
  • Mot:Lien, nom masculin
    • Un lien peut être un texte, un bouton ou une images
sur lesquels il est possible de cliquer pour accéder à
une page html ou à un contenu par exemple. 
Tu auras plus d'informations en cliquant sur ce lien.
  • Mot:Onglet, nom masculin
    • Elément d'une fenêtre sur lequel vous pouvez cliquer
ou passer votre souris pour accéder à un documebt 
ou à des fonctions et plus encore.
Le signe "+" te permet de cliquer et d'ouvrir
un nouvel onglet
  • Mot:Page, nom féminin
    • Une page web contient les informations
d'un site internet.
L'utilisation de cette page web est très ergonomique
